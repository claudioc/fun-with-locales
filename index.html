Fun with Locale Formatters

What is this page all about?

This is a simple web page that showcases the Intl object, using it to format dates, times, numbers, and currencies according to the a specified locale. Just plain HTML and JavaScript, no 3rd-party libraries are used which also proves that you can do a lot with the web platform itself.

Browser default locale is and currency default is EURO
